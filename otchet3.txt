Отчет по лабораторной работе №3
Настройка локального маршрутизатора для переадресации пакетов между vlan
Цель работы: Научиться настраивать подинтерфейсы маршрутизатора для организации связи между различными виртуальными сетями
1.	Создание подинтерфейсов для каждого vlan
En // переключение в привилегированной режим
Conf t // переключение в режим конфигурирования
Int fa0/0.2 //создание подинтерйфеса для 2 vlan
Enc dot 1Q 2 // включение инкапсуляции для 2 vlan
Ip ad 192.168.2.254 255.255.255.0 //присвоение ip адреса подинтерфейсу

Int fa0/0.3 
Enc dot 1Q 3
Ip ad 192.168.3.254 255.255.255.0 
…
2.	Включение физического интерфейса 
Int fa0/0
No shutdown
3.	Настройка транка на коммутаторе 
Выбор интерфейса, соединённого с маршрутизатором 
Int fa0/5
Switchport mode trunk
Switchport  trunk allowed vlan 2-5
4 Настройка шлюзов на компьютерах 
Для VLAN 2 192.168.2.254
Для VLAN 3 192.168.3.254
Для VLAN 4 192.168.4.254
Для VLAN 5 192.168.5.254
